
# Shreya Engineering Works

A stunning, modern React + Tailwind CSS website for an engineering shop, featuring heavy animations and a beautiful, eye-catching design.

## Features
- Proprietor & Manager contact info
- Address and services offered
- Special note on hydraulic press axle straightening
- Animated, modern UI with framer-motion and react-awesome-reveal
- Responsive and mobile-friendly

## Media
- Place your real shop/media images in `public/media/` (currently uses a placeholder for axle-press.jpg)

## Development
1. Install dependencies (if not already):
	```sh
	npm install
	```
2. Start the development server:
	```sh
	npm run dev
	```
3. Open [http://localhost:5173](http://localhost:5173) in your browser.

## Custom Animations
Custom slow spin and reverse spin utilities are defined in `src/custom-animations.css` and imported in `src/index.css`.

## Replace Media
Replace `public/media/axle-press.jpg` with a real image for production use.
